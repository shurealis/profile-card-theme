# profile-card-theme
A profile card with theme color changing feature. Created using HTML5 , Tailwind CSS and Javascript.

## Initiate Package.JSON
```
npm init -y
```

## Install tailwindCSS
```
npm install -D tailwindcss
npx tailwindcss init
```
